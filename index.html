Cambios de HDD a SSD

  1. Dar limpieza al CPU
  2. Clonar el hdd en el ssd de no encontrar errores en el arranque de Windows
  3. Cada maquina debe contar con 3 usuarios:
    • Administrador: S1s73m@s
    • Sistemas: S1s73m@s
    • Car-One: carone (Este perfil no puede cambiar la contraseña y estará limitado a autorización de administrador).
  4. Alimentar inventario verificando y corrigiendo de ser necesario las columnas: Departamento, nombre de equipo, ip, tipo de disco y observaciones.

En caso de encontrar errores en el arranque de Windows

  1. Dar limpieza al CPU
  2. Cargar Windows desde 0
  3. Cada maquina debe contar con 3 usuarios:
    • Administrador: S1s73m@s
    • Sistemas: S1s73m@s
    • Car-One: carone (Este perfil no puede cambiar la contraseña y estará limitado a autorización de administrador).
  4. Instalar programas que se comparten mediante esta liga: Google Drive
  5. Alimentar inventario verificando y corrigiendo de ser necesario las columnas: Departamento, nombre de equipo, ip, tipo de disco y observaciones.

Configuración de Red y Host

¿Cómo ejecutar el script de configuración de red?

  1. Rellena los campos del formulario con la información de red deseada.
  2. Haz clic en el botón "Generar Script de Configuración de Red" para descargar el archivo configurar_red.ps1.
  3. Busca el archivo descargado en tu carpeta de "Descargas".
  4. Importante: Haz clic derecho sobre el archivo configurar_red.ps1 y selecciona "Ejecutar como administrador" para que los cambios de red y nombre de host tengan efecto.

Nota: El script intentará configurar el primer adaptador de red que encuentre activo. Si tienes múltiples adaptadores, verifica el script antes de ejecutarlo.

¿Cómo ejecutar el script de creación de usuarios?

  1. Haz clic en el botón "Generar Script de Creación de Usuarios" para descargar el archivo crear_usuarios.ps1.
  2. Busca el archivo descargado en tu carpeta de "Descargas".
  3. Haz clic derecho sobre el archivo crear_usuarios.ps1 y selecciona "Ejecutar con PowerShell".
  4. Si aparece una advertencia de seguridad, permite que el script se ejecute.
  5. Alternativa (Modo Administrador):
    1. Abre el Menú Inicio, busca "PowerShell", haz clic derecho y selecciona "Ejecutar como administrador".
    2. En la ventana de PowerShell, navega a la carpeta donde descargaste el script. Ejemplo: cd $HOME\Downloads
    3. Ejecuta el script escribiendo: .\crear_usuarios.ps1

Nota: El script está diseñado para ser seguro, pero siempre es una buena práctica revisar el contenido de los scripts antes de ejecutarlos.
